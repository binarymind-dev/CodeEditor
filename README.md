# CodeEditor

A feature-rich C++ Code Editor built with Qt6 for Windows.

## Features
- Syntax Highlighting (C/C++)
- Line Numbers
- Multiple Tabs
- Dark / Light Theme
- Find and Replace
- Compile and Run C/C++ code
- Auto Save
- Font Size Control

## Built With
- C++17
- Qt6 (Widgets, GUI)
- MinGW

## How to Build
cmake .. -G MinGW Makefiles -DCMAKE_PREFIX_PATH=C:/Qt/6.11.1/mingw_64
mingw32-make -j4
